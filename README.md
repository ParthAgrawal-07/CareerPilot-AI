### Intelligent Career Guidance Platform

CareerPilotAI is a fully developed AI-powered career guidance platform designed to help students make informed academic stream decisions after completing 10th grade.  
The application uses conversational artificial intelligence to analyze user interests, preferences, and dislikes, delivering personalized career recommendations along with financial and professional insights.

---

## About CareerPilotAI
Choosing the right academic stream is a critical decision that impacts a studentâ€™s future education and career path. CareerPilotAI simplifies this process by offering an interactive, data-driven, and accessible career counseling experience.

The platform combines conversational AI with structured career data to provide clear, personalized guidance in real time.

---

## Key Capabilities
- AI-driven conversational career counseling
- Personalized stream recommendation based on user interests
- Real-time analysis of education costs
- Career role insights with average salary expectations
- Interactive chat-based user experience
- Cloud-hosted and scalable deployment

---

## How It Works
1. Users interact with the AI counselor through a chat interface  
2. The system analyzes user inputs to understand interests and dislikes  
3. The AI determines readiness for recommendation  
4. A suitable academic stream is suggested  
5. Relevant cost and career data is displayed for informed decision-making

## Technology Overview
- **Language:** Python  
- **Web Framework:** Streamlit  
- **AI Engine:** Google Gemini (Flash Model)  
- **Database:** SQLite  
- **Deployment:** Streamlit Cloud  
- **Version Control:** Git & GitHub  

---

## AI Model Choice
CareerPilotAI uses the **Gemini Flash** model to balance performance and cost efficiency.  
This model offers fast responses and strong conversational reasoning, making it ideal for real-time user interaction without requiring paid billing tiers.

---

## Security & Reliability
- API keys are securely managed using environment-based secrets
- No sensitive user data is stored permanently
- Cloud deployment ensures reliability and scalability
- Rate-limit handling prevents service interruptions

---

## Live Deployment
CareerPilotAI is deployed on Streamlit Cloud and is accessible via a public URL.  
The deployment pipeline securely integrates application code with environment secrets, ensuring safe and reliable execution.
